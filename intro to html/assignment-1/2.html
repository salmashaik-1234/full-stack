b tag

The b html element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the boldface element, and most browsers still draw the text in boldface.

i tag

The i tag defines a part of text in an alternative voice or mood. The content inside is typically displayed in italic.

u tag

the u tag is used to represents some text that is unarticulated and styled differently from normal text,such as misspelled words or proper names in chinese text. the content inside is typically displayed with underline.

select tag

the slect tag is used to crete a drop-down list.The select element is most often used in a form,to collect user input.

button tag

the button tag defines a clickable button. Inside a button element you can put text.

hr tag

The hr tag defines a thematic break in a html page.The hr element is most often displayed as a horizental rule that is used to seperate content(or defne a change) in an html page.

br tag

The br tag inserts a single line break.The br tag is useful for br tag is an empty tag which means that it has no end tag.
